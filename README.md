# Financial accounts of Slovenia

This repository contains the rendered files for the overview tables of financial accounts and balance sheets of Slovenia available at this [link](https://gasperploj.github.io/financial-accounts-slovenia/).

The tables give an overview of:
* financial assets and liabilities by economic sector
* financial transactions by economic sector
* who-to-whom tables

The report is updated every Monday at around 11:30 CET via GitHub Actions.
